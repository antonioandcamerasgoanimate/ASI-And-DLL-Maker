# ASI-And-DLL-Maker
A ASI And DLL Maker for easy create scripts.
An all-in-one developer utility designed to instantly generate clean, ready-to-compile Cplusplus/Csharp ASI and DLL mod templates for Grand Theft Auto V. Whether you are a seasoned developer looking to skip the boilerplate setup or a beginner jumping into GTA V modding, this tool sets up your Visual Studio environment in seconds.

No more manually configuring inclusion directories, linking ScriptHookV SDKs, or messing with project properties. Just input your mod details, and get a flawless, structured project ready for your custom code.

Key Features
Instant ASI & DLL Generation: Create standardized Cplusplus templates specifically structured for GTA V runtime injection.

Pre-Configured ScriptHookV Integration: Automatically sets up the scriptMain loop, keyboard hooks, and essential native calling structures.

Prerequisites & Requirements
To compile the generated projects, you will need:

Visual Studio 2022 (with Desktop Development with Cplusplus installed)

ScriptHookV SDK (Ensure you point your project to your local ScriptHookV include/lib paths has installed on setup)

Installation:
Cilck on setup.exe
install Visual Studio 2022 (with Desktop Development with Cplusplus installed).
